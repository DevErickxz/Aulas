Tipos int e float
int -> inteiro -> 5, 6, 7, -5, -6, -7
o tipo int ele representa qualquer numero
positivo ou negativo int sem sinal é positivo

float -> numero com ponto flutuante(casa decimal) -> 5.5, 6.5
o tipo float representa qualquer numero
positivo ou negativo com ponto flutuante

a função type retorna o tipo de dado
